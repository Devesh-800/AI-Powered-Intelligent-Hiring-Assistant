# 🤖 AI Powered Intelligent Hiring Assistant

An AI-powered Hiring Assistant built using **Machine Learning, NLP, RAG, and Generative AI** to automate resume screening and candidate evaluation.

---

## 🚀 Features

- 📄 Resume Classification
- 📊 Resume vs Job Description Matching
- 🎯 Skill Gap Analysis
- 📈 Final Hiring Score
- 🤖 AI-Powered Feedback using OpenRouter
- 💬 RAG-Based Career Chatbot
- 📚 Knowledge Base Retrieval (CSV)
- 📥 Downloadable PDF Report

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Scikit-learn
- Sentence Transformers
- OpenRouter API
- Pandas
- NumPy
- PDFPlumber
- ReportLab
- RAG Pipeline
- Explainable AI
